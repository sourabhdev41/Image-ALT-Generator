# Image ALT Generator

Image ALT Generator is a lightweight WordPress plugin that automatically generates SEO-friendly ALT text from image filenames.

The plugin works completely offline without using AI, external APIs, or third-party services. It follows the native WordPress admin interface and is designed to be fast, secure, and easy to use.

## Features

- Generate ALT text from image filenames
- Bulk generate ALT text for existing images
- Skip existing ALT text (optional)
- Simple SEO score for generated ALT text
- WooCommerce image support
- Native WordPress admin UI
- Lightweight and fast
- No external APIs or AI required

## Requirements

- WordPress 6.0 or later
- PHP 7.4 or later

## Installation

### Recommended Installation

1. Go to the **Releases** section of this repository.
2. Download the latest **plugin ZIP** (for example, `image-alt-generator.zip`).
3. In your WordPress dashboard, navigate to **Plugins → Add New → Upload Plugin**.
4. Select the downloaded plugin ZIP.
5. Click **Install Now**.
6. Activate the plugin.

## Important

**Do not install the ZIP file downloaded from GitHub's "Code → Download ZIP" or the automatically generated "Source Code (ZIP)" archive.**

Those archives contain the complete project source code and **are not installable WordPress plugins**.

If you downloaded the repository source code, you have two options:

- Use the **plugin ZIP** included in the repository (if available).
- Or package only the plugin folder into a ZIP file before uploading it to WordPress.

Only the bundled plugin ZIP should be uploaded through the WordPress Plugin Installer.

## Usage

1. Go to **Media → ALT Generator**.
2. Choose whether to skip images that already have ALT text.
3. Click **Generate ALT Text**.
4. Wait for the bulk generation process to finish.
5. Review the generated ALT text and SEO scores.

## WooCommerce Support

If WooCommerce is installed, the plugin also processes:

- Product featured images
- Product gallery images
- Product variation images

No additional configuration is required.

## Compatibility

- WordPress 6.0+
- PHP 7.4+
- WooCommerce (optional)
